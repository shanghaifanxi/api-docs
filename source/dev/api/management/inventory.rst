主机清单
======================

-----------------------------------------------------------------------------------------------------------------------
- 获取主机清单


1、主机清单查询
-----------------------------------------------------------------------------------------------------------------------


**请求方式：**   GET

**请求地址：**   /api/control/inventory/


**Content-Type：**
::

    application/x-www-form-urlencoded —— 表示通过表单方式提交
    application/json —— 表示传入数据为json格式字符串


**GET查询参数：**

+------------------------+------------+------------+----------------------------------------+
|**参数**                |**数据类型**|**是否必须**|**说明**                                |
+------------------------+------------+------------+----------------------------------------+
| id                     | int        | 是         | inventory id                           |
+------------------------+------------+------------+----------------------------------------+
